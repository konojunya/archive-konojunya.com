md-preview-electron

Markdown Editor

DESCRIPTION

最近流行りのFacebook製のViewライブラリであるReactと、Node.jsを使って
デスクトップのアプリケーションをWEBの言語で作れるElectronを使い、markdown editorを作りました。

エディタ部分にはAceを使い、previewのCSSはgithubと同じハイライトを使い、
プレビューはReactiveに変換していきます。変換にはmarked.jsを使いました。

ボタン部分の配置はGoogleのMaterial Design Guideに従い配置し色はこのエディタの変換では出てこない、
白に映える明るめのピンクを採用しました。
